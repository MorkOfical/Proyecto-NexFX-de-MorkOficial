## Desarrollo

El repositorio incluye herramientas básicas de calidad de código:

```bash
pnpm install
pnpm run lint          # ESLint
pnpm run format:check  # Prettier (solo comprobar)
pnpm run format:write  # Prettier (aplicar formato)
```
